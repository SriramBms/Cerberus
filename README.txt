This is a Chrome Extension to keep track of Wikipedia articles on your watchlist.

1. Install the .crx file.
2. Just click the extension icon on the brower to see any recent changes to the watchlist items.
3. You can change the number of items displayed and whether you want to see multiple revisions to
     the same article in the settings/options page.

Known Issues and work-arounds.

1. Sometimes, Clicking on the button throws a blank popup with just the 'Settings' link. If this happens,
   click on the 'Settings' link and change one or two settings, save it and try again.
